# UN Official Meeting Records

Which UN organs produce official meeting records, and in what format.

## Record Types

- **PV (Procès-Verbal / Verbatim Records)** — full first-person, word-for-word transcripts. Document symbol suffix: `/PV.`
- **SR (Summary Records)** — third-person condensed summaries with numbered paragraphs. Document symbol suffix: `/SR.`

No organ receives both PV and SR (GA Rule 58(a)).

## Rules of Procedure References

- **GA**: Rule 58(a) of A/520/Rev.21 — verbatim records for plenary and 1st Committee only
- **SC**: Provisional Rules of Procedure, Chapter IX, Rules 48-57
- **ECOSOC**: E/5715/Rev.2
- **HRC**: Decision 47/114 (2021) — summary records for action-taking meetings from 48th session
